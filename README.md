# Smartphone Store

Web application to browse, search and manage a catalog of smartphones: product list with real-time search, product detail with storage/color configuration, and a persistent shopping cart.

> Full documentation (architecture, decisions, scripts) will be completed as the project evolves.

## Requirements

- Node.js 18.18+ (see `.nvmrc`)

## Quick start

```bash
npm install
npm run dev     # development mode (unminified assets)
npm run build   # production build (concatenated + minified assets)
npm start       # serve production build
```
